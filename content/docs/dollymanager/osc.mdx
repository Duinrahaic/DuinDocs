# OSC Interface

You can control dolly manager through your avatar using OSC. VRCDollyManager will listen for updates from the following avatar parameters.

| Address       | Type    | Descripton                   |
|---------------|---------|------------------------------|
| VDM_Delay_Inc | Boolean | Increases Delay              |
| VDM_Delay_Dec | Boolean | Decreases Delay              |
| VDM_Play      | Boolean | Plays Currently Loaded Dolly | 
| VDM_Stop      | Boolean | Stops Currently Loaded Dolly |

Additionally, a pre-made version of the avatar menu can be purchased [here](https://duinrahaic.booth.pm/items/6930308)
