# General Usage

### Playing
Click the `Play` Button to play the dolly in VRChat
![](https://cdn.phz.my/i/Screenshot_20251212_160657.png)

<Accordions type="single">
  <Accordion title="You can change the delay on the dolly animation by using the dial next to the play button.">![](https://cdn.phz.my/i/Screenshot_20251212_160853.png)</Accordion>
</Accordions>

### Searching
Search and select the dolly you want to manage.
![](https://cdn.phz.my/i/Screenshot_20251212_150849.png)

### Editing
Click the `Edit` Button to rename or export the selected dolly.
![](https://cdn.phz.my/i/Screenshot_20251212_150948.png)

### Previewing
Click the `Preview` tab to view the selected dolly in 3D.
![](https://cdn.phz.my/i/Screenshot_20251212_151123.png)

### Tagging
Click the `Tags` tag to add or remove tags from the dolly.
![](https://cdn.phz.my/i/Screenshot_20251212_151256.png)

### Loading
Click the `Load` button to load the selected dolly into VRChat.

If you want to load the dolly relative to your current position, enable the `Relative` button.
![](https://cdn.phz.my/i/Screenshot_20251212_151530.png)

<Callout type="warn">
  Loading a will only be available if OSC is enabled and connected.
</Callout>

### Exporting

1. Select the dolly you want to export
2. Click the `Edit` button
3. Click the `Copy To Clipboard` button
4. Share the copied file with your friends or paste it somewhere else.

### Importing

1. Drag and drop the dolly file into the Dolly Manager window.
2. The file will be automatically imported and added to the list
    <Callout>If the dolly is already in the list, it will be updated with the information from the new file.</Callout>
