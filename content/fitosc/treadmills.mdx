# Supported Treadmills

This table shows all currently supported and tested treadmills for FitOSC and VRTI applications.

<TreadmillsTableServer />

## Legend

- **BLE**: Bluetooth Low Energy support
- **FTMS**: Fitness Machine Service (standard Bluetooth protocol)
- **Speed**: Read/Write capabilities for speed control
- **FitOSC**: Compatible with FitOSC application
- **VRTI**: Compatible with VRTI application


## Adding Your Treadmill

If you have a treadmill that's not listed here, please contribute to the database by submitting a pull request or opening an issue on GitHub.
