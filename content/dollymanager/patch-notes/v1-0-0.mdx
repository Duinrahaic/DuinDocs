**Release Date:** March 29, 2025

## Features
- Labeling functionality for tagging and renaming dollies
- 3D preview capability before VRChat loading
- Non-invasive file management approach
- Lightweight, dependency-free application
- OSC Query integration for seamless support
- Automatic monitoring for newly created dollies
- Content sharing via clipboard file copying
