## Searching
The search bar at the top of the page can be used to filter through dolly names, aliases, and tags.
![](/docs/dollymanager/Screenshot_20251212_150849.png)

## Loading
The `Load` button imports the selected dolly into VRChat.

If you want to load the dolly relative to your current position, enable the `Relative` button.
![](/docs/dollymanager/Screenshot_20251212_151530.png)

<Callout type="warn">
  Loading a will only be available if OSC is enabled and connected.
</Callout>

## Playing
Once a dolly is loaded into VRChat you can press the `play` button which starts the dolly animation after the specified delay.

  <Accordion title="You can change the delay on the dolly animation by using the dial next to the play button.">![](/docs/dollymanager/Screenshot_20251212_160853.png)</Accordion>
</Accordions>

![](/docs/dollymanager/Screenshot_20251212_160657.png)

## Editing
The `Edit` Button allows you to rename or export the selected dolly.
![](/docs/dollymanager/Screenshot_20251212_150948.png)

## Previewing
The `Preview` tab allows you to view the selected dolly in 3D.
![](/docs/dollymanager/Screenshot_20251212_151123.png)

## Tagging
The `Tags` tag allows you to add or remove tags from the dolly.
![](/docs/dollymanager/Screenshot_20251212_151256.png)

## Exporting
1. Select the dolly you want to export
2. Click the `Edit` button
3. Click the `Copy To Clipboard` button
4. Share the copied file with your friends or paste it somewhere else.

## Importing
1. Drag and drop the dolly file into the Dolly Manager window.
2. The file will be automatically imported and added to the list
    <Callout>If the dolly is already in the list, it will be updated with the information from the new file.</Callout>
